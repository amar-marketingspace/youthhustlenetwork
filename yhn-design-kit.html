10 — Usage Rules
✓ Do
  • Use Bebas Neue for all display headings and hero text
  • Keep orange (#FF5C00) as the dominant accent — never dilute with other brand colours
  • Use yellow (#FFD000) only for secondary accents, badges, and the H in YHN
  • Apply angled clip-path to all CTA buttons
  • Use DM Sans for all body copy at 1.7+ line-height
  • Keep backgrounds dark — #0A0A0A, #111, #161616 only
  • Use grid-line textures behind hero text for depth
  • Keep section labels in Syne 700, uppercase, 3px letter-spacing, orange colour
  • Use the marquee strip on orange background to separate major sections
✗ Don't
  • Don't use white or light backgrounds on any digital material
  • Don't use Inter, Roboto, or system fonts — brand fonts only
  • Don't use more than two accent colours per layout
  • Don't stretch or distort the YHN logo
  • Don't use orange text on orange backgrounds
  • Don't use rounded corners on elements — YHN is sharp and angular
  • Don't use gradients on buttons — flat solid colours only
  • Don't reduce yellow to a background fill — it's an accent only
  • Don't change the Y-H-N colour split — Y is orange, H is yellow, N is orange
